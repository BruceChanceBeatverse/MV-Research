## Motion Prompts

### Singing Motions

**Ballad/Slow Song**
```
Stands mostly still, subtle head movements, gentle hand gestures near chest, closes eyes during emotional peaks
```

**Pop Performance**
```
Sways side to side, bounces slightly to rhythm, arm movements during chorus, big smile
```

**Powerful Vocals**
```
Leans forward slightly, holds microphone with both hands, intense eye contact, strong facial expressions
```

**Intimate Performance**
```
Minimal movement, soft expressions, slight head tilts, gentle swaying
```

### Rap Motions

**Aggressive/Hype**
```
Gestures emphatically with both hands, nods head sharply, points at camera, leans forward with intensity
```

**Smooth Flow**
```
Subtle hand movements, relaxed stance, slight head bob, cool confident expression
```

**Storytelling Rap**
```
Varied gestures matching narrative, expressive hands, shifts weight side to side, direct eye contact
```

**Battle Rap**
```
Sharp pointing gestures, aggressive facial expressions, steps toward camera, strong stance
```

### Performance Energy Levels

**Low Energy (Emotional/Sad)**
```
Minimal movement, downcast eyes occasionally, soft gestures, standing still with slight sway
```

**Medium Energy (Conversational)**
```
Natural hand gestures, moderate head movements, comfortable stance, friendly expressions
```

**High Energy (Upbeat/Party)**
```
Big arm movements, bouncing to beat, wide smile, head movements to rhythm, animated expressions
```

**Peak Energy (Drop/Climax)**
```
Throws hands up, intense facial expression, leans back then forward, big gestures
```

## Common Mistakes to Avoid

### Avatar Prompts

❌ "Avatar that is happy and energetic"
✅ "Avatar smiling and bouncing, wearing bright colors"

❌ "Professional looking person"
✅ "Avatar in business suit, standing in modern office"

❌ "Cool rapper vibe"
✅ "Avatar in streetwear with chains, urban background"

### Motion Prompts

❌ "Dances around energetically"
✅ "Bounces to beat, moves arms side to side"

❌ "Shows emotion"
✅ "Closes eyes, gentle smile, touches chest"

❌ "Performs like a professional"
✅ "Confident stance, strong gestures, direct eye contact"

### Voice Prompts

❌ "Sounds good"
✅ "Mid-pitch, warm tone, confident delivery"

❌ "Like a famous singer"
✅ "Powerful vocals, emotional delivery, rich tone"

## Testing Strategy

1. **Start Simple**
   - Basic avatar description
   - Simple motion prompt
   - Generate and evaluate

2. **Add Details**
   - Enhance avatar with specific clothing
   - Add environment details
   - Refine motion descriptions

3. **Generate Variations**
   - Try 3-5 different versions
   - Adjust one element at a time
   - Compare results

4. **Find What Works**
   - Save successful prompts
   - Build personal prompt library
   - Reuse and modify for future videos
