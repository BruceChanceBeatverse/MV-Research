## Detailed Cost Analysis

### HeyGen Avatar IV Pricing

#### Base Plan Costs

**Creator Plan:**
- **Monthly:** $29/month
- **Annual:** $24/month ($288/year)
- **Includes:** 10 minutes (600 seconds) Avatar IV credits/month

**Cost per second (base plan only):**
```
$29 / 600 seconds = $0.048 per second
$24 / 600 seconds = $0.040 per second (annual)
```

#### After Monthly Credits Exhausted

**GenCredit Pack Add-on:**
- **Monthly:** $15/month for 300 GenCredits
- **Annual:** $150/year for 3,600 GenCredits
- **Conversion:** 1 GenCredit = 2 seconds of Avatar IV video

**Cost per second (add-on credits):**
```
$15 / 600 seconds = $0.025 per second
$150 / 7,200 seconds = $0.021 per second (annual)
```

#### Real-World Cost Examples

**Scenario 1: Light User (5 videos/month @ 60 sec each)**
- Total: 300 seconds/month
- Cost: $29/month (Creator plan covers it)
- **Per-second cost: $0.097/sec**

**Scenario 2: Regular User (15 videos/month @ 60 sec each)**
- Total: 900 seconds/month
- Base plan: 600 seconds ($29)
- Additional: 300 seconds = 150 GenCredits ($7.50)
- Total cost: $36.50/month
- **Per-second cost: $0.041/sec**

**Scenario 3: Heavy User (50 videos/month @ 60 sec each)**
- Total: 3,000 seconds/month
- Base plan: 600 seconds ($29)
- Additional: 2,400 seconds = 1,200 GenCredits ($60)
- Total cost: $89/month
- **Per-second cost: $0.030/sec**
